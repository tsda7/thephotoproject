# Front-End Code for dantschanz.ch

all the js and css files required

## Development

The code is developed in Typescript, placed in `src/*` and then compiled to `lib/` using the Typescript-Compiler (`tsc`).
To start developing, download and install node.js and enable the yarn package manager ([see link](https://yarnpkg.com/getting-started/install))
Then install all dependencies with the command `yarn install` and compile the code using `yarn run build`
